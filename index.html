Example icon buttons

Icons taken from Bootstrap Xtra and modified some css.

A range of icons can be added by adding a class of .icon and any one of the provided icon classes.

You can also add the class .right to put the icons on the right side of the text and .alternative class for extra icons color.

Using other icon colors, just add a new css class:

.btn.icon.newcolor:before {
        color: #FFFF00;
}

Code example

        <button class="btn icon search">Default</button>

        <button class="btn primary icon alternative search">Primary</button>

        <button class="btn info icon alternative search">Info</button>

Sticky Options

Name default description
p ptc position (ptc = p - position, t - top, c - center) options:ptl,ptc,ptr,pbl,pbc,pbr
sp fast animation speed
x true delay before hiding sticky (ms) or false to disable (make sticky)

Simple Example

$.sticky('Hello Sticky!');
Say Hello!

Sticky with Icon and custom position

$.sticky('<b>Position!</b><p>Position bottom-center..</p>', 'icon arrowdown', {p:'pbc'});
Position bottom-center

Show some element inside Sticky

$('blockquote').sticky(); //default position, no icon.
Show some element

Permanent until closed

$.sticky('<b>Sticky!</b><p>Sticky message goes here..</p>', 'icon remove', {x:false,p:'ptr'});
Sticky message.
